# CheckMarkCell

The CheckMarkCell is a sub component to be used inside a row.

## Usage

```jsx
import React from 'react';
import { CheckMarkCell } from 'terra-table';

<CheckMarkCell
  isSelected={isSelected}
  onSelect={onSelection}
  isSelectable
  key="unique-key"
/>
```
